# Flunguage — программное обеспечение для обучения иностранным словам

**Flunguage** — это десктопное приложение для изучения иностранных слов с использованием интерактивных карточек, различных режимов тренировки и рекомендаций, формируемых на основе нейросетевого анализа. Приложение позволяет создавать собственные словари, отслеживать прогресс обучения и настраивать интерфейс под свои предпочтения (светлая/тёмная тема, размер шрифта).

### Функционал

- Управление словарями: создание, редактирование и удаление словарей с возможностью добавления слов из общего списка или создания новых.
- Управление словами: добавление, редактирование, удаление слов с указанием перевода, транскрипции, языка и уровня сложности.
- Тренировка: три режима обучения (mixed — смешанный, input — ручной ввод, choice — выбор из вариантов) с настройкой источника слов, типа слов (new/good/medium/bad) и количества.
- Статистика: отображение общей успеваемости (точность, среднее время ответа), диаграмма распределения слов по уровням усвоения, таблица сложных слов и список последних тренировок с возможностью удаления записей.
- Рекомендации от нейросети: предложение слов для изучения на основе семантической близости с использованием векторных представлений word embeddings.
- Настройки: переключение между светлой и тёмной темой, изменение размера шрифта с сохранением параметров в конфигурационном файле.
- Навигация: удобное переключение между экранами (Словарь, Тренировка, Статистика) через нижнюю панель.

### Запуск приложения

#### Предварительные требования

Перед началом работы убедитесь, что на вашей системе установлены:

1. Python (версия 3.10 или выше);
2. pip (менеджер пакетов Python, устанавливается вместе с Python);
3. виртуальное окружение (рекомендуется, но не обязательно).

*Приложение работает на операционных системах Windows, Linux и macOS.*

#### Установка Python

Перейдите на официальный сайт [Python.org](https://www.python.org/downloads/)

Скачайте и установите последнюю стабильную версию Python 3.x для вашей операционной системы.

Проверьте установку, выполнив в терминале (командной строке):
```sh
python --version
pip --version
```

#### Распаковка архива

Распакуйте архив `DPFlunguage.zip` в удобную директорию. Откройте эту директорию в терминале или в IDE (например, PyCharm, VS Code).

#### Создание виртуального окружения (рекомендуется)

Для изоляции зависимостей проекта создайте виртуальное окружение:

**Windows:**
```sh
python -m venv venv
venv\Scripts\activate
```

**Linux / macOS:**
```sh
python3 -m venv venv
source venv/bin/activate
```

#### Установка зависимостей

В терминале, находясь в директории проекта, выполните команду:

```sh
pip install -r requirements.txt
```

Все необходимые зависимости будут установлены. Основные из них:
- PyQt6 — для графического интерфейса;
- matplotlib — для построения диаграмм статистики;
- qtawesome — для иконок;
- numpy (используется для работы с векторными представлениями).

#### Запуск приложения

После установки всех зависимостей запустите приложение командой:

```sh
python main.py
```

Приложение откроется в отдельном окне. При первом запуске автоматически создаётся файл базы данных `app.db` и конфигурационный файл `config.json` с настройками по умолчанию.

#### Настройка нейросетевых рекомендаций (опционально)

Для работы модуля рекомендаций `WordRecommender` в директории проекта должны находиться предварительно обученные векторные представления слов. При их отсутствии модуль продолжает работу, но рекомендации могут быть недоступны или ограничены.

### Структура проекта

```
DPFlunguage/
├── main.py                 # точка входа в приложение
├── requirements.txt        # список зависимостей Python
├── app.db                  # файл базы данных SQLite (создаётся автоматически)
├── config.json             # файл конфигурации (тема, размер шрифта)
├── repositories/           # репозитории для работы с БД
├── services/               # бизнес-логика приложения
├── ui/                     # все компоненты интерфейса
│   ├── widgets/            # виджеты экранов (словарь, тренировка, статистика)
│   ├── dialogs/            # диалоговые окна
│   └── styles/             # стили кнопок и элементов
└── utils/                  # вспомогательные модули
```

### Стек технологий

- **Python 3.10+** — язык программирования;
- **PyQt6** — фреймворк для создания графического интерфейса;
- **SQLite** — встроенная база данных (через `sqlite3`);
- **matplotlib** — построение диаграмм статистики;
- **qtawesome** — векторные иконки;
- **numpy** — вычисления для нейросетевых рекомендаций.
```
